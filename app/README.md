### 📰 News

Pet-приложение для чтения новостей с возможностью подписки на интересующие темы.

Проект создан для изучения современного Android-стека и многомодульной архитектуры.

📱 Приложение позволяет:

🔥 Просматривать актуальные новости

📋 Подписываться на новости по интересующим темам

🔄 Автоматически обновлять новости в фоновом режиме

📖 Читать полные статьи в браузере

📤 Делиться статьями с друзьями

🛠️ Технологический стек

- UI Jetpack Compose (Material 3)
- Архитектура Clean Architecture + MVVM
- DI Dagger Hilt
- База данных Room
- Сеть Retrofit
- Асинхронность Kotlin Coroutines + Flow
- Фоновые задачи WorkManager
- Навигация Compose Navigation
- Загрузка изображений Coil
- Модульность Многомодульный Gradle проект

## 🚀 Запуск проекта

### 1. Получение API ключа

Для работы приложения необходимо получить API ключ на [NewsAPI](https://newsapi.org/):

1. Перейдите на [NewsAPI](https://newsapi.org/)
2. Зарегистрируйтесь (бесплатный тариф позволяет делать до 100 запросов в день)
3. Скопируйте ваш API ключ из личного кабинета

### 2. Настройка keystore.properties

Создайте файл `keystore.properties` в **корне проекта** (рядом с `settings.gradle.kts`)

и добавьте ваш API ключ (сам ключ в кавычках):

```properties
# keystore.properties
NEWS_API_KEY="your_api_key_here"
```

### 3. Сборка и запуск

Откройте проект в Android Studio, дождитесь синхронизации Gradle (нажмите "Sync Now" или иконку
слоника 🐘),

затем нажмите Run ▶️ для запуска приложения на эмуляторе или реальном устройстве.
